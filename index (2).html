02Windows Server licensing uplift

Dev/Test bills Windows VMs at Linux rates with no licence meter, so repricing usage alone understates the CSP cost. This section adds the licence delta back for Windows VMs.

04What the numbers can't show

  • No SLA on Dev/Test. Dev/Test offers carry no financially-backed SLA. Azure Plan (CSP) restores standard SLAs — worth real money if these workloads matter.
  • Production is prohibited on Dev/Test. If any of this estate is quietly running production, moving to CSP is a compliance fix, not just a commercial choice.
  • Visual Studio subscription dependency. Dev/Test rates require active Visual Studio subscribers. CSP removes that dependency and its per-seat cost.
  • Marketplace & purchase charges are excluded from the repricing — they bill identically on either offer.
  • SQL Server on Dev/Test also bills without the SQL licence meter. If the estate includes SQL VMs, the true CSP cost is higher than shown unless AHB covers SQL too.
  • Reservations & savings plans are available on Azure Plan and can close the gap further — this tool compares against PAYG retail as the conservative case.
